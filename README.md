## Run Locally

Clone the project

```bash
  git clone https://github.com/Pinqua/whatsapp-web-clone.git
```

Go to the project directory

```bash
  cd whatsapp-web-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
